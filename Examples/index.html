Selecting element(s)

Here are some examples of how an element can be selected

// Select by element

$$('html').html();

// Select by id

$$('#id').html();

// Select by class

$$('.class').html();

// Select by reference

$$(this).html();

Page Ready

ready(callback) calls the callback when the page is done loading.

// Select by element

$$().ready(function(){

alert('The page is loaded!');

});
